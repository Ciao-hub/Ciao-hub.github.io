    • 树莓派配置Trojan

    1. 下载trojan客户端

    wget https://github.com/sfantree/trojan-armv6/raw/master/trojan_1.14.1-git5074793_armhf.deb

    1. 安装trojan

    dpkg -i trojan_1.14.1-git5074793_armhf.deb

    1. 编辑配置文件

    vim /usr/local/etc/trojan/config.json

    1. 配置文件内容，编辑后保存退出
    {

  "run_type": "client",    #Trojan客户端类型

  "local_addr": "127.0.0.1",  #本地IP地址

  "local_port": 5566,     #本地端口

  "remote_addr": "trojan.evergrowingsh.com",  #服务器地址

  "remote_port": 443,              #服务器Trojan端口

  "password": [                 #服务器Trojan密码

​    "Yunhe789"

],
    1. 运行Trojan

    trojan #运行后保持终端窗口开启

    1. 设置Trojan代理

    a) 下载代理软件

    apt-get install proxychains

    b) 编辑配置文件

    vim /etc/proxychains.conf

    c) 配置文件修改内容，编辑后保存退出

    #在最后的ProxyList注释默认的socks代理，socks4 前加#注释

    socks5 127.0.0.1 5566 # 在最后的ProxyList里加入Trojan的代理设置；5566为trojan本地端口号，与前面trojan配置文件里local_port一致

    d) 测试是否代理成功

    proxychains curl -4 ip.sb #若显示服务器IP地址，则代理成功

    1. 用Trojan代理运行Python爬虫

    proxychains python3 xxx.py
